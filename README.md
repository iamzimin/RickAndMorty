# RickAndMorty

An application for viewing heroes, locations, episodes of the Rick and Morty universe

Stack: Kotlin, Clean Architecture, Jetpack Compose, Coroutine, Ktor, Room, Dagger Hilt, MVVM, Pagination, JUnit, Mockito
